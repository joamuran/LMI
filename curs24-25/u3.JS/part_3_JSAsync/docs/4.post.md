# Peticions amb `POST`

Les peticions HTTP de tipus POST ens permeten enviar dades a un servidor o una API. Per a això, caldrà fer ús d'un segon paràmetre de fetch, anomenat *init*, amb el qual li podem proporcionar a fetch un objecte amb diferents opcions per
inicialitzar la petició, entre elles, el tipus de petició amb `method`, però també les capçaleres de la petició (`headers`) o el contingut del cos (`body`).

```js
fetch(url, init);
```

Veiem-ho amb un exemple:

```js
async function afegirUsuari() {
    const resposta = await fetch('https://jsonplaceholder.typicode.com/users', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json' // Especifica que enviem JSON
        },
        body: JSON.stringify({ 
            name: "Joan", 
            email: "joan@example.com" 
        })
    });

    const dades = await resposta.json();
    console.log("Usuari afegit:", dades);
}

afegirUsuari();
```

Com veiem, fem una petició web a l'URL `https://jsonplaceholder.typicode.com/users`, i el bloc *init* conté el següent:

```js
 {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json' // Especifica que enviem JSON
    },
    body: JSON.stringify({ 
        name: "Joan", 
        email: "joan@example.com" 
    })
}
```

Ací s'especifica:

* Paràmetre `method`, indiquem el tipus de petició, an aquest cas, de tipus **POST**.
* Paràmetre `headers`, on indiquem diferents propietats per a la capçalera de la petició. En aquest cas, especifiquem `Content-Type` com a `application/json`.
* Paràmetre `body`, amb el contingut que enviem al servidor, en aquest cas, una cadena de text que conté un JSON. Per a això fem ús del mètode estàtic `JSON.stringify()`, que rep un JSON i ens el retorna en forma de cadenade text.
